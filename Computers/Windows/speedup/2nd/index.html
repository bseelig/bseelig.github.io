Follow the steps using 360 Total Security and Clean Master:

<-- Home

1. Open 360 Total Security 2. Open Tool Box 3. Open Game Booster 4. Click on the boost tab
5. This list shows all the background programs that are running on your computer. Check the list to make sure you know what is running. Check the items that are using a lot of memory.
6. Then click boost now. Note: When you repeat these steps click restore and then start at step 5 Now use Clean Master to finish up. Note: You only need to do this once 1. Open Clean Master 2. On the top right side, click the 3 striped line menu then open Settings 3. On the left side, click the Junk Clean tab 4. Scroll down to Auto Clean and turn it on. 5. Check the items that you want to clean. Recommended Cleaning: Windows, Browser, Malware, and Software 6. Click on the Privacy Clean tab and scroll down to the Browser Anti-tracking then turn it on 7. Check the items that you want to clean. Recommended Cleaning: Search History, Downloaded, History, Typed URL's, and Multimedia Now you should be all good with cleaning -<-/c>
